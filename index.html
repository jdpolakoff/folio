Fishwrap is a front-end news briefing application that pulls headlines from hundreds of news sites. Swipe through stories with a "Tinder"-like carousel.

NOTE: This project is in beta mode and only currently accessible through http protocal, which causes some unreliability on page refreshes. If you get a 404 error, please navigate to the homepage (http://fishwrap.herokuapp.com) and try to reach your desired page again. After a production-level release of the next version of the API, the site will be fully launched and available through https protocal.

TECHNOLOGIES: React, jQuery, News API, Git.
